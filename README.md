# Pro Talent Connect Frontend

This repository contains the Pro Talent Connect frontend application.

## Project Location

The Vite + React application source lives in:

- `frontend/`

## Getting Started

1. Go to the app folder:
   - `cd frontend`
2. Install dependencies:
   - `npm install`
3. Start development server:
   - `npm run dev`

## Build

From `frontend/` run:

- `npm run build`

## Test

From `frontend/` run:

- `npm run test`
